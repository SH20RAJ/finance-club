# Finance Club BIT Mesra - Official Website

## Overview

Welcome to the official repository for the Finance Club of BIT Mesra (FinPro / FINWHIZ). This website is designed to reflect the club's mission of fostering financial literacy, market awareness, and economic insight among students.

The project features a "Premium Fintech" aesthetic, utilizing deep greens, golds, and glassmorphism to create a sophisticated and immersive user experience. It serves as the central hub for our flagship event **FinStreet**, workshops, and student resources.

## ğŸ Getting Started

This project uses [Bun](https://bun.sh) as the package manager and runtime.

### Prerequisites

- Install Bun: `curl -fsSL https://bun.sh/install | bash`

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/your-username/finance-club.git
    cd finance-club
    ```

2.  Install dependencies:

    ```bash
    bun install
    ```

3.  Run the development server:

    ```bash
    bun dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
